PERGUNTAS FREQUENTES

Preciso ter algum conhecimento prévio para me tornar piloto?

Não! O ebook foi desenvolvido justamente para quem está começando do zero e explica todos os passos iniciais.

Qual a idade mínima para iniciar a formação como piloto?

No Brasil, é possível iniciar os estudos teóricos a qualquer momento, mas para obter a licença de piloto privado é necessário ter 17 anos completos.

Quanto tempo leva para se formar como piloto comercial?

O tempo varia conforme a disponibilidade do aluno, mas em média leva de 18 a 24 meses para completar toda a formação.
